# m-cli docs

Documentation for `m-cli` — the engine-neutral `m` toolchain CLI.

## Folders

- **`guides/`** — how-to guides.
  - [Claude Code Permissions Guide](guides/claude-code-permissions-guide.md) — clean-room blueprint for configuring Claude Code permissions (user + per-repo settings).
- **`memory/`** — auto-memory store (durable gotchas + cross-repo facts; see [`memory/MEMORY.md`](memory/MEMORY.md) for the index).

There is no generated `modules/` reference here (that is a stdlib-repo artifact),
and no live tracker or `design/` notes at present — add them under the standard
folder set if a workstream needs one.
